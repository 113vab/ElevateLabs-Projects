# 🛡 Project 1 — ASTRAVA

## Cyber Threat Detection & Monitoring Platform

ASTRAVA is a cybersecurity platform designed to monitor network traffic, identify suspicious activities, and provide administrators with real-time visibility into network security events through an interactive web dashboard.

The platform demonstrates practical applications of packet analysis, threat detection, security monitoring, and visualization techniques commonly used in Security Operations Centers (SOC).

###  Key Features

-  Live Network Packet Monitoring
-  Threat Detection Engine
-  Security Analytics Dashboard
-  Traffic Visualization
-  Security Alert Management
-  Live Activity Feed
-  Packet Inspection
-  Security Event Logging
-  Responsive User Interface

###  Technologies Used

- Python
- Flask
- Scapy
- SQLite
- HTML5
- CSS3
- JavaScript

###  Skills Demonstrated

#  Project 2 — VIONA

## Secure Real-Time Messaging Platform

VIONA is a modern secure messaging platform that enables real-time communication between users while providing an intuitive and responsive user experience.

The application was designed using a relational database architecture and implements modern messaging features similar to contemporary communication platforms.

###  Key Features

-  Secure User Authentication
-  Contact Management
-  Private Conversations
-  Real-Time Messaging
-  Online Presence
-  Typing Indicators
-  Message Delivery Status
-  Read Receipts
-  Conversation Search
-  Responsive Interface
-  PostgreSQL Database
-  Socket.IO Real-Time Synchronization
